# SystemFileManagerX Documentation

Welcome to the **SystemFileManagerX** documentation!

SystemFileManagerX is a modern C++17 console-based file manager for Windows and Linux. It provides a fast, keyboard-driven interface for navigating directories, managing files, and exploring your system in a dual-pane layout.
